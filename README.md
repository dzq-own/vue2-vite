# binfo-vue-tpl

> 开发模式推荐使用vite运行，打包必需使用vue-cli

- public
- src
    - api 后端API封装
    - assets 样式及资源
    - common 其他
    - components 全局UI组件
    - router 路由配置
    - store Vuex
    - utils 工具函数封装
    - views 页面及业务组件
- .editorconfig
- .gitignore
- package.json
- vue.config.js: Vue-Cli 配置文件
- vite.config.js: Vite2配置文件

### Quick Start

```
// 使用Vue-Cli运行
npm run serve
```

```
// 使用vite运行
npm run dev:vite
```
```
// 使用vue-cli打包
npm run build
```
